# BookShoppingCartMVC
A web-based Book Shopping Cart application built using ASP.NET Core MVC and SQL Server.


## Project Overview
**Book Shopping Cart MVC** is a full-stack web application built using **ASP.NET MVC Framework** and **SQL Server**.  
It allows users to browse, add, and purchase books online with an easy-to-use interface.  
The admin panel helps manage book inventory, categories, and customer orders.



## Features
- User Registration & Login (with authentication)
- Browse books by category
- Add books to shopping cart
- Order management system
- Database integration using Entity Framework
- Responsive UI with Bootstrap


## Technologies Used
| Layer | Technology |

| Frontend | HTML, CSS, Bootstrap |
| Backend | ASP.NET MVC 5 (C#) |
| Database | SQL Server |
| IDE | Visual Studio 2022 |


## Installation Steps

### Step 1: Clone this Repository
git clone https://github.com/your-username/BookShoppingCartMVC.git

### Step 2: Open Project in Visual Studio
Go to File → Open → Project/Solution
Open BookShoppingCartMvc.sln

### Step 3: Restore Dependencies
Tools → NuGet Package Manager → Restore Packages

### Step 4: Update Database Connection 
Edit the file appsettings.json and set your SQL Server name:

"ConnectionStrings": {
  "DefaultConnection": "Server=YOUR_SERVER_NAME;Database=BookDB;Trusted_Connection=True;"
}

### Step 5: Apply Database Migrations
Open Package Manager Console and run:  update-database

### Step 6: Build and Run
